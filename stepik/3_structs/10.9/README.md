# Task 3.10.9

> Дано трехзначное число. Найдите сумму его цифр.

Формат входных данных
На вход подается трехзначное число.

Формат выходных данных
Выведите одно целое число - сумму цифр введенного числа.

```terminal
Sample Input:
253

Sample Output:
10
```
